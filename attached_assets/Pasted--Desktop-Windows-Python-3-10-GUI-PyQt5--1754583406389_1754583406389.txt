Проект: Программа для расчёта строительных смет (Desktop, Windows)
Язык: Python 3.10+
GUI: PyQt5 (или PySide6)
Требования: современный и приятный визуально интерфейс (стиль "финтех/ERP"), экспорт PDF/Excel, хранение проектов.

1. Описание задачи
Создать десктопное приложение для автоматизации расчёта строительных смет. Программа должна позволять пользователю:

Добавлять/редактировать/удалять виды работ (с объемами, расценками и привязкой к материалам).

Вести справочник материалов с текущими ценами и единицами измерения.

Автоматически рассчитывать количество и стоимость материалов на основе заданных норм расхода для каждой работы.

Формировать сводную смету: стоимости работ, материалов, общая сумма.

Экспортировать результат в Excel и PDF (с оформлением).

Сохранять и загружать проекты (файловая система, JSON или SQLite).

Интерфейс — современный, чистый, приятный для глаз, с акцентом на удобство (см. "ERP style desktop app", темная/светлая тема, таблицы, диалоги, крупные кнопки, опционально логотип).

2. Структура данных
Работа
Наименование работы

Единица измерения

Себестоимость

Цена за единицу

Объем (вводит пользователь)

Список материалов с нормой расхода на единицу работы

Материал
Наименование

Единица измерения

Цена за единицу

Ссылка на поставщика (опционально)

Вес/примечание (опционально)

Смета/Проект
Название проекта

Дата создания/изменения

Перечень выбранных работ (с объемами)

Итоги по работам и материалам

3. Функционал интерфейса
Главное окно:

Список работ (таблица), возможность добавлять/редактировать/удалять работы

Для выбранной работы — отображение связанных материалов и норм расхода

Быстрый ввод объемов по работам, пересчет по кнопке

Таблица итоговых материалов по всему проекту

Кнопки: “Добавить работу”, “Добавить материал”, “Рассчитать”, “Экспорт PDF/Excel”, “Сохранить проект”, “Открыть проект”

Импорт/экспорт работ и материалов (CSV/XLSX)

Современная визуализация:

Красивые таблицы, цветовые акценты, иконки, аккуратный layout

Опционально: выбор темы (темная/светлая), вставка логотипа

Всплывающие окна для редактирования/добавления записей

Сохранение всех данных в файл (JSON или SQLite)

Поддержка нескольких проектов

4. Логика расчета
Для каждой работы:

Стоимость работы = Объем × Цена за ед.

Для каждого материала:

Кол-во = Объем работы × норма расхода

Стоимость материала = Кол-во × Цена за ед.

Итоги:

Сумма по работам, сумма по материалам, общая сумма по проекту

5. Экспорт
Экспорт всей сметы в Excel (xlsx), с разбивкой по работам и материалам

Экспорт в PDF (по оформленному шаблону, итоги по проекту)

6. Требования к коду
Чистая архитектура: разделение данных, логики, интерфейса

Комментарии и PEP8-стиль

Готовность к расширению (например, добавление новых типов расчетов)

Установка зависимостей через requirements.txt

7. Визуальные предпочтения
Современный стиль: крупные кнопки, красивые таблицы, цветовые акценты

Опционально: фирменные цвета и логотип (добавить параметрами)

Интерфейс интуитивно понятный, минимум лишнего
